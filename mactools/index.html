这里纪录一些我常用的Mac工具, 因为我是后台开发, 所以更多的是命令行工具。

Item2

Item2是加强版的终端, 自带分屏工具, 2.9+以上支持终端显示图片, 还可以热键呼出, 非常好用!

  • 官网

使用截图

HomeBrew

包管理工具, 程序员必备, 包管理是*nix系统的一大利器, Mac的HomeBrew基于Github, 开源。

  • 官网
  • Github

Oh-My-ZSH

shell里面zsh最好, zsh里面oh-my-zsh则是最佳。oh-my-zsh提供快速跳转、自动补全, 美丽的交互界面,功能十分强大。

  • 官网
  • Github

MuxiVim

嗯, MuxiVim 是我自己的vim配置, 自我感觉用的比较爽, 提供目录树导航、类函数tag、文件快速搜素、好看的airline、多个主题 以及各种快捷键绑定。没有自动补全, 结合Item2的分屏, 十分好用!

  • Github

Tmux

Item2的分屏虽然好用, 但是不能保存状态, Tmux是分屏的一大神奇, 尤其是服务器端开发, 有了Tmux随时掉线、随时ssh连接, 且保存状态, 而且可以很方便的查询和连接窗口。

  • 我的tmux配置

WhatPortIs

Python神奇的杰作!

  • github

PtPython

个人感觉最好用的Python解释器界面,没有之一(bpython也可以就是不太稳定)

多行编辑

支持vi, emacs 多种模式!

Snap

可以快速呼出常见的应用 command + [1, 2, 3, 4...]

总结

Mac就是开发上的神器, 从系统到体验到工具, 所以很欢迎更多的人入Mac:) 唯一遗憾的就是Mac上少了折腾的乐趣。
